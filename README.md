# Mail.tm

Generate and manage temporary email addresses directly from Raycast using the free Mail.tm API service.

## Features

- **Quick Generate**: Instantly create temporary emails with user-friendly random addresses
- **Customize**: Create emails with custom address, password, and domain selection
- **Inbox Viewer**: Read your temporary emails directly in Raycast
- **Local Storage**: Saved emails persist with automatic expiration after 7 days
- **Secure**: Passwords are masked by default with secure copy actions
- **Clean UI**: Organized interface with keyboard shortcuts for all actions

## Usage

### Quick Generate
Select "Quick Generate" to instantly create a random temporary email. The email address is automatically copied to your clipboard.

### Customize Email
Select "Customize Email" to create a temporary email with:
- Custom email address (local part)
- Custom password
- Domain selection (if multiple domains available)

Leave fields empty to generate random values.

### View Saved Emails
Select "View Saved Emails" to:
- See all your temporary emails
- View mailbox and read messages
- Copy email address or password
- Delete emails when done

## Keyboard Shortcuts

- `⌘ R` - Refresh
- `⌘ C` - Copy email/message text
- `⌘ ⇧ C` - Copy password/sender email
- `⌃ X` - Delete email
